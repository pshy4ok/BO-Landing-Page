## B&O Landing Page

A responsive, production-ready landing page for Bang & Olufsen that showcases premium audio and smart home products.
The goal of this project is to present the brand’s products in a clean, minimal Scandinavian-inspired layout with:
- **Clear product storytelling**
- **Easy navigation between categories**
- **Fast, mobile-first experience**

The page is suitable as a portfolio piece and as a boilerplate for future marketing or product landing pages.

## Technologies Used

- **HTML5**: Semantic and accessible markup for better SEO and screen-reader support.
- **SCSS (Sass)**: Modular styles with a BEM-like structure for maintainable and scalable CSS.
- **Parcel**: Zero-config bundler for assets, SCSS compilation, and local development server.

## Getting Started

Follow these steps to run the project locally on your machine.

### 1. Clone the repository

```bash
git clone https://github.com/pshy4ok/BO-Landing-Page.git
cd BO-Landing-Page
```

### 2. Install dependencies

Using npm:

```bash
npm install
```

Or, if you prefer Yarn:

```bash
yarn install
```

### 3. Run the project locally

Using npm:

```bash
npm start
```

Or with Yarn:

```bash
yarn start
```

After the dev server starts, the app will be available at the address printed in your terminal.

## Features

- **Responsive layout**: Optimized for desktop, tablet, and mobile viewports with adaptive typography, spacing, and grid.
- **Product showcase**: Highlighted hero section and featured products with imagery, descriptions, and pricing emphasis.
- **Category navigation**: Separate sections for Portable speakers, Earphones, and Smart Home devices for quick exploration.
- **Brand storytelling**: “Timeless, for 50 years and counting” content block to communicate the brand’s heritage and values.
- **Contact form**: Simple contact form with basic validation for name, email, and message fields to simulate lead capture.
- **Clean codebase**: Structured SCSS partials and reusable components to make future changes or redesigns easier.
